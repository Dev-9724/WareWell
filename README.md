# WareWell

Smart wardrobe outfit recommendation system.

## Project
CST3990 Undergraduate Individual Project  
Middlesex University London

## Author
Dev Patel

## Tech Stack
Backend:
- Python
- FastAPI
- MongoDB

Frontend:
- Vue 3
- Vite

External API:
- OpenWeather API

## Features
- Context-aware outfit recommendation
- Hard constraint filtering
- Multi-criteria scoring
- Weather integration
- Feedback learning
- Evaluation dashboard

## System Architecture

Vue Frontend  
↓  
FastAPI Backend  
↓  
Constraint Engine  
↓  
Outfit Generator  
↓  
Scoring Engine  
↓  
MongoDB Database  
↓  
Weather API
